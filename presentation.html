Agents communicate via Redis. Agent 1 stores output, Orchestrator triggers Agent 2 with the job ID. Results assembled in a background thread.

Flask API · Gunicorn (4 workers) · Nginx reverse proxy · Docker Compose · Rate limiting on /api/generate (2 req/s)
